Yumeko Jabami [夢子蛇喰] เป็นตัวเอกที่มีพรสวรรค์ในการค้นพบกลโกงในเกมแห่งโอกาส ในช่วงเริ่มต้นของการทำงานหญิงสาวได้พบกับซูซุยซึ่งตลอดทั้งเรื่องกลายเป็นเพื่อนที่ดีที่สุด ของตัวละครเธอมีความกล้าหาญอย่างมากเธอมักจะพบว่าตัวเองเดิมพันด้วยมูลค่าสูงและเธอ ไม่สนใจที่จะเสียเงินโดยพิจารณาจากรายละเอียดเพียงเล็กน้อยสำหรับอารมณ์ที่เธอรู้สึกเมื่อ ต้องเสี่ยงในเกมซึ่งสำหรับเธอนั้นสำคัญที่สุดนอกจากนี้ยูเมโกะในช่วงฤดูกาลแรกต้องเผชิญ หน้ากับสมาชิกสภานักเรียนซึ่งเป็นผู้ที่เก่งที่สุดในโรงเรียนเนื่องจากพวกเขาใช้วิธีการที่เข้ม งวดในระหว่างการเดิมพัน แน่นอนว่าเธอไม่ใช่ตัวละครที่อยู่ยงคงกระพัน
